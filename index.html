소음일기

개인정보처리방침 · Privacy Policy

1. 기본 원칙

소음일기는 사용자의 개인 식별 정보를 수집하지 않습니다. 모든 데이터는 사용자의 기기에만 저장됩니다.

소음일기(이하 "앱")는 층간소음 피해자가 소음 및 대응 활동을 기록하고 관리할 수 있도록 도와주는 서비스입니다. 본 방침은 앱이 수집하는 정보와 그 사용 방법에 대해 설명합니다.

2. 수집하는 정보

앱은 아래의 정보만을 수집하며, 모든 정보는 사용자의 기기 내에만 저장됩니다.

  • 소음 기록 — 소음 유형, 발생 시간, 출처, 위치, 지속시간 등 사용자가 직접 입력한 정보
  • 신체/정신 피해 기록 — 수면 방해, 두통, 불안, 약 복용 여부, 우울감 등 사용자가 직접 입력한 건강 관련 정보
  • 대응 활동 기록 — 대응 방법, 일시, 결과 등 사용자가 직접 입력한 정보
  • 녹음 파일 — 사용자가 직접 녹음한 오디오 파일 (기기 내 저장)
  • 녹화 파일 — 사용자가 직접 녹화한 영상 파일 (기기 내 저장)

3. 수집하지 않는 정보

앱은 아래의 정보를 수집하지 않습니다.

  • 이름, 이메일, 전화번호 등 개인 식별 정보
  • 위치 정보 (GPS)
  • 기기 고유 식별자
  • 광고 식별자
  • 로그인 계정 정보

4. 데이터 저장 및 보안

모든 데이터는 100% 사용자의 기기 내에만 저장됩니다. 외부 서버로 전송되지 않습니다.
  • 모든 기록 데이터는 기기 내 로컬 저장소(AsyncStorage)에 저장됩니다.
  • 녹음, 녹화 파일은 기기 내 파일 시스템에 저장됩니다.
  • 각 기록에는 SHA-256 해시값이 부여되어 기록 시점의 무결성을 보호합니다.
  • 앱을 삭제하면 모든 데이터가 함께 삭제됩니다.

5. 민감정보 처리

신체/정신 피해 기록(수면 방해, 두통, 약 복용 여부 등)은 사용자가 직접 입력한 경우에만 수집되며, 기기 내에만 저장됩니다. 외부로 전송되지 않습니다.

해당 정보는 층간소음 피해 증거 자료로 활용하기 위한 목적으로만 사용되며, 사용자가 직접 내보내기 기능을 통해 공유하는 경우 외에는 외부로 유출되지 않습니다.

6. 요청하는 권한

  • 마이크 — 층간소음 녹음 및 데시벨 측정을 통한 증거 수집 목적
  • 카메라 — 소음 발생 현장 녹화를 통한 증거 수집 목적
  • 저장소 — 녹음, 녹화 파일의 기기 내 저장 목적
  • 알림 — 주간/월간 소음 기록 요약 알림 발송 목적 (사용자가 허용한 경우에만 작동)

모든 권한은 사용자가 해당 기능을 사용할 때만 요청되며, 거부하더라도 다른 기능은 정상적으로 사용할 수 있습니다.

7. 제3자 제공

앱은 사용자의 데이터를 제3자에게 제공, 판매, 공유하지 않습니다.

단, 사용자가 직접 내보내기(Export) 기능을 사용하여 외부로 공유하는 경우는 사용자 본인의 선택에 의한 것입니다.

앱 내 이웃사이센터 메뉴에서 한국환경공단 외부 사이트(floor.noiseinfo.or.kr)로 연결될 수 있으며, 해당 외부 사이트의 개인정보처리방침은 별도로 적용됩니다. 앱은 해당 사이트에서 발생하는 정보 수집에 관여하지 않습니다.

8. 아동 개인정보 보호

앱은 만 14세 미만 아동을 대상으로 하지 않으며, 아동의 개인정보를 의도적으로 수집하지 않습니다.

9. 방침 변경

개인정보처리방침이 변경될 경우 앱 내 공지 또는 본 페이지를 통해 사전에 안내드립니다. 변경된 방침은 공지 후 7일이 경과한 시점부터 효력이 발생합니다.
